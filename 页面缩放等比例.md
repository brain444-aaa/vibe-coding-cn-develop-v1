强制执行流程（严格按步骤输出）

第一步：第一性原理·问题根源深度拆解（300-400 字）

从浏览器渲染底层逻辑、CSS 单位本质、表格布局特性三个维度，精准拆解我页面的核心问题：

1. 浏览器缩放的本质：不是修改 CSS 数值，而是拉伸/压缩逻辑像素与物理像素的映射比例（1 逻辑 px 对应 1.5/0.5 物理 px），这是缩放错乱的核心底层逻辑；

2. 现有页面的核心矛盾：固定像素（px）定义的子元素与相对单位（%/auto）定义的父容器不匹配，缩放时父容器空间动态变化，子元素 px 数值固定，无法同步适配，导致位移、变形；

3. 表格换行的根本原因：未启用table-layout: fixed固定布局，表格默认table-layout: auto，列宽由内容动态分配，缩放时列空间被挤压，浏览器强制内容换行破坏布局；

4. 无法等比缩放的关键：缺乏统一的相对单位基准，各元素尺寸规则混乱，缩放时无统一缩放锚点，各元素比例失衡。

第二步：第一性原理·修复方案推导（200-250 字）

基于底层逻辑，推导根治方案的核心原理与落地规则：

1. 统一缩放基准：采用rem相对单位，以根元素（html）字体大小为全局基准，所有尺寸（宽、高、间距、字体、边框）基于 rem 定义，确保缩放时所有元素同步等比变化；

2. 隔离外部干扰：为页面套一层固定宽度根容器（基于 rem 定义），配合居中与横向滚动，隔离浏览器窗口缩放对内部布局的直接影响，避免父容器空间动态变化引发的错位；

3. 锁定表格布局：强制表格使用table-layout: fixed固定布局，配合white-space: nowrap禁止内容换行，通过overflow: hidden+ellipsis处理超长内容，确保表格缩放时结构稳定；

4. 统一布局规范：使用 Flex/Grid 布局替代传统浮动，所有间距、尺寸均用 rem 定义，消除布局浮动带来的缩放错位风险。

第三步：可落地执行·完整修复方案（分模块输出）

1. 全局 rem 基准配置（必加 CSS）
  /* 根元素字体基准：16px，所有 rem 基于此计算 */
  html {
   font-size: 16px;
  }
  /* 全局盒模型统一，避免尺寸计算偏差 */
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
    }
2. 页面根容器配置（必加 CSS）
  /* 页面根容器：固定宽度（80rem=1280px），居中，超出横向滚动 */
  .page-root-container {
   width: 80rem;
   max-width: 100%;
   margin: 1rem auto;
   padding: 1.5rem;
   background-color: #fff;
   overflow-x: auto; /* 窗口小于容器时出现横向滚动，保护内部布局 */
  }
3. 表格锁定配置（必加 CSS）
  /* 表格外层容器：防止表格撑破根容器 */
  .table-scroll-container {
   overflow-x: auto;
   margin: 1.5rem 0;
   border: 1px solid #e5e6eb;
   border-radius: 0.25rem;
  }
  /* 表格核心配置：固定布局+不换行+超长省略 */
  table {
   width: 100%;
   table-layout: fixed; /* 锁定列宽，不随内容变化 */
   border-collapse: collapse;
   font-size: 1rem; /* 16px，基于 rem 基准 */
  }
  th, td {
   padding: 0.75rem 0.5rem;
   text-align: left;
   border-bottom: 1px solid #e5e6eb;
   white-space: nowrap; /* 禁止内容换行 */
   overflow: hidden;
   text-overflow: ellipsis; /* 超长内容显示省略号 */
  }
  th {
   background-color: #f5f7fa;
   font-weight: 600;
  }
4. 通用布局配置（按需替换原有浮动布局）
  /* 通用 Flex 布局：适配组件排列，避免浮动错位 */
  .flex-row {
   display: flex;
   align-items: center;
   gap: 1rem; /* 间距基于 rem，同步缩放 */
  }
  .flex-col {
   display: flex;
   flex-direction: column;
   gap: 1rem;
  }
  /* 通用间距：所有间距统一用 rem */
  .mt-1 { margin-top: 1rem; }
  .mb-1 { margin-bottom: 1rem; }
  .p-1 { padding: 1rem; }
